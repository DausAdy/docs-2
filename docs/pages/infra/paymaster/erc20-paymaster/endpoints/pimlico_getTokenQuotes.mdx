# pimlico_getTokenQuotes

This method returns the tenatative token exhange rates used by the ERC-20 Paymaster.

## Request

```json
{
    "jsonrpc": "2.0",
    "method": "pimlico_getTokenQuotes",
    "params": [
        {
            "tokens": [
                "0x6b175474e89094c44da98b954eedeac495271d0f",
                "0x2260fac5e5542a773aa44fbcfedf7c193bc2c599",
                "0x514910771af9ca656af840dff83e8264ecf986ca"
            ] // list of tokens to get quotes for
        },
        "0x0000000071727De22E5E9d8BAf0edAc6f37da032", // entrypoint address
        "0x1" // chainId
    ],
    "id": 1
}
```

## Response

:::warning
If a token in the request input is not supported by the ERC-20 Paymaster, the request will still succeed but it will not be included in `quotes` array in the response.
:::
